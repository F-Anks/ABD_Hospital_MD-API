# API Hospital MD

Este proyecto proporciona una interfaz para la gestión y generación de datos hospitalarios altamente realistas.

## Pruebas de Inyección de Datos
Se han realizado pruebas exhaustivas de carga para validar la escalabilidad del sistema:

- **Escenarios Probados:** 10 (Volumen, Género, Edad, Condiciones Médicas).
- **Total de Datos Inyectados:** 135,109 registros.
- **Documentación Detallada:** Puedes consultar el informe completo y las capturas de pantalla en [README de Test](test/README.md).
